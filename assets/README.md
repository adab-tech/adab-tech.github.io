# Assets Directory

## CV File
Please add your CV PDF file here with the name `Adamu_Danjuma_CV.pdf` to enable the download functionality in the portfolio.

The CV link in `longblog.html` is configured to point to `assets/Adamu_Danjuma_CV.pdf`.

## Instructions
1. Add your CV PDF file to this directory
2. Name it `Adamu_Danjuma_CV.pdf`
3. The download button on the portfolio will automatically work

Alternatively, you can update the link in `longblog.html` to point to any other location where your CV is hosted.
