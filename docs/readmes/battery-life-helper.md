# Battery Life Helper

Windows PowerShell utility to extend laptop battery life through configurable power profiles and diagnostics.

## Overview

Battery Life Helper packages practical Windows tuning scripts for students and developers who need longer runtime without manual registry hunting.

## Features

- Profile-based power tweaks
- Readable CLI output for Windows 10/11
- Lightweight, no installer required

## Quick start

```powershell
git clone https://github.com/adab-tech/battery-life-helper.git
cd battery-life-helper
# Review scripts · run from elevated PowerShell if required
```
