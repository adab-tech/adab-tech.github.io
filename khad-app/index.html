About Khad App

Khad App is a healthcare communication demonstration platform designed to address language barriers in medical settings. Built with accessibility and safety as core principles, it helps patients describe their symptoms clearly and assists healthcare providers in understanding multilingual patient inputs.

⚕️ Important Notice

This application is a demonstration tool only. It does not provide medical advice, diagnoses, or treatment recommendations. All medical decisions must be made by qualified healthcare professionals. If you have a medical emergency, call emergency services immediately.

Design Principles

  • Patient-Centered: Simple, accessible interfaces for users of all technical levels
  • Mobile-First: Responsive design optimized for smartphones and tablets
  • WCAG Compliant: Accessible to users with disabilities
  • Privacy-Focused: API keys stored locally, no server-side data storage
  • No Diagnostics: Strictly focused on communication clarity, not medical assessment

Technical Architecture

Built as a static web application compatible with GitHub Pages, Khad App uses:

  • • TailwindCSS for responsive styling
  • • Vanilla JavaScript for client-side logic
  • • Web Speech API for voice input
  • • AI Router Module for backend abstraction
  • • LocalStorage for configuration persistence

Use Cases

  • • Community health centers serving multilingual populations
  • • Telemedicine platforms needing symptom clarification
  • • Medical education and training scenarios
  • • Research in healthcare communication barriers
