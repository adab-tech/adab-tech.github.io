## Overview

A comprehensive billing and inventory management system designed for healthcare facilities, featuring patient management, inventory tracking, and financial reporting capabilities.

## Key Features

- **Patient Management**: Complete patient records and billing history
- **Inventory Control**: Real-time tracking of medical supplies
- **Financial Reports**: Automated billing and revenue analytics
- **Multi-user Support**: Role-based access control
- **Cloud Deployment**: Scalable and secure infrastructure

## Technical Implementation

Built with Django for robust backend functionality, PostgreSQL for reliable data storage, and Bootstrap for responsive user interface design.
